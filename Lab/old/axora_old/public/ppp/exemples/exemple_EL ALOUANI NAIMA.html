Plan Personnalisé de Prévention (PPP) — EL ALOUANI NAIMA

Document partagé à l’issue de l’entretien


Grande Pharmacie de Tassigny — Fréjus
Professionnel effecteur : Pharmacien
Date : 18/11/2025
Âge : 65 ans

INSIGHTS DP : constipation chronique (Transipeg 5,9 g, Eductyl®, Normacol®), préparation colique récente (Moviprep®), bas/chaussettes de compression, antalgiques (dont piroxicam), L‑thyroxin 25–50 µg, colchicine 1 mg, vitamine D 50 000 UI, kit de dépistage colorectal remis 07/2025.



Bilan biologique — synthèse & décisions

  • Leucocytes urinaires élevés sans bactériurie et hématies urinaires légères : tableau compatible avec leucocyturie stérile. Conduite : pas d’antibiotique en l’absence de symptômes urinaires ; refaire un ECBU sur jet médian si signes apparaissent ou si l’anomalie persiste ; rechercher cause non infectieuse (irritation, lithiase, contamination).
  • Fonction rénale : DFG ≈ 70 mL/min/1,73 m², stable. Catégorie G2 ; si albuminurie absente, risque faible et pas de maladie rénale chronique avérée. Action : maintenir hydratation, éviter AINS per os ; surveiller DFG/albuminurie au suivi.
  • Thyroïde : TSH et hormones dans la cible. Action : poursuivre L‑thyroxin, prise à jeun 20–30 min avant le petit‑déjeuner, éloignée du fer/calcium (2–4 h).
  • Helicobacter pylori : négatif. Action : RAS.
  • Foie : ALAT et GGT légèrement élevées mais en amélioration. Action : prioriser hygiène de vie (poids, activité, alcool nul ou minimal), revue médicamenteuse ; si persistance/aggravation ou > 3× N, avis médical. Les « drainages hépatiques » en phytothérapie peuvent exposer à des hépatotoxicités : privilégier un suivi médical plutôt qu’une automédication.




Thème prioritaire Freins / points de vigilance Conseils & modalités Ressources & intervenants
1) Transit & constipation chronique
Objectif : confort digestif durable, limiter les lavements
  • Usage répété de lavements/suppositoires.
  • Facteurs favorisants : hydratation/fibres insuffisantes ; médicaments (AINS).
  • Macrogol (PEG) = 1re intention ; titrer la dose jusqu’à selles souples, puis sevrage progressif quand le transit est régulier.
  • Lavements/suppositoires : usage ponctuel uniquement (risque d’irritation/dépendance).
  • Mode de vie : hydratation ≥ 1,5 L/j, 25–30 g de fibres/j, marche quotidienne, horaire toilette après repas.
  • AINS per os (piroxicam) : à éviter si possible (risque rénal et digestif chez 65+) ; préférer paracétamol pour douleurs.
  • Pharmacie : plan de titration du PEG, éducation sevrage lavements, repérage automédications à risque (AINS).
  • Médecin traitant / gastro‑entérologue : avis si échec > 4–6 sem, signaux d’alerte, examen colique si indiqué.
2) Prévention fragilité, chutes & santé veineuse
Objectif : autonomie, mobilité, soulager les symptômes veineux
  • Compression veineuse en cours ; peau fragile.
  • Risque de déconditionnement (≥ 65 ans).
  • Activité : viser 150–300 min/sem d’activité modérée + renforcement 2 j/sem + exercices d’équilibre.
  • Compression : enfiler le matin, retirer le soir ; vérifier l’absence de plis ; hydrater la peau le soir ; associer marche et élévation des jambes.
  • Domicile sécure : éclairage, suppression tapis instables, chaussures fermées antidérapantes.
  • Pharmacie : contrôle taille/pose des bas, essai enfile‑bas, repérage ICOPE (mobilité, vision, audition, humeur, cognition, nutrition).
  • Médecin / kinésithérapeute : programme équilibre/renforcement, bilan chute.


Suivi & prochaines étapes

  • Biologie : refaire ECBU si symptômes urinaires ou persistance des anomalies ; suivi DFG/albuminurie au contrôle annuel ; re‑contrôle ALAT/GGT si besoin.
  • Vaccinations 65+ (si non faites) : pneumocoque PCV20 (1 dose), zona Shingrix (2 doses M0–M2/6), grippe annuelle.
  • Dépistages organisés : colorectal (test tous les 2 ans, 50–74 ans) ; sein (mammographie tous les 2 ans, 50–74 ans).
  • Point officine : J+14 (transit, réduction lavements) ; J+30 (activité/équilibre, peau sous compression, statut vaccins/dépistages).

Ce PPP ne remplace pas un avis médical. Les adaptations thérapeutiques et décisions vaccinales/dépistages se font avec le médecin traitant.
